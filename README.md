# 不要太早把自己变成一种身份

这是一本仍在生长中的小册子，讨论一个创造者、研究者或转型中的人，如何不急着给自己定位，而是先靠近那些反复抓住自己的现象、困惑和问题。

它介于个人方法论、思想札记和创造者手册之间。

## 核心问题

- 在身份之前，是什么先抓住了我？
- 一个问题如何从现象中慢慢长出来？
- 学科、组织、作品如何帮助我靠近问题？
- 作品和组织又如何反过来塑造我？
- 身份如何成为脚手架，而不是房子？

这不是一本关于“如何找到职业定位”的手册，而是一份关于“如何延迟定位、深入问题、让作品自然长出来”的写作实验。
