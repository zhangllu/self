一个多专题作品仓库。每件专题作品讨论一个独立主题，但共享同一种写作和研究方式——从现象出发，不急着定位。

## 专题作品1：身份系列

讨论一个人如何不急着给自己定位，而是先靠近那些反复抓住自己的现象、困惑和问题。身份是沉淀，不是起点。

**状态**：八篇文章 + 序言 + 结语，持续修订中

## 专题作品2：高阶认知

学习历史上智者的高阶认知方式——借用他人经验扩大感知半径。

**状态**：5 篇章节

[进入专题作品 →](books/专题作品2-高阶认知/README.md)

## 专题作品3：焦虑地图

讨论焦虑作为一种现代人的普遍经验——它不是需要被消灭的问题，而是一种可以被理解、被回应、被转化的信号。
