一个多小册子仓库。每本小册子讨论一个独立主题，但共享同一种写作和研究方式——从现象出发，不急着定位。

## 学习智者的高阶认知：主动适应复杂社会

讨论一个创造者、研究者或转型中的人，如何不急着给自己定位，而是先靠近那些反复抓住自己的现象、困惑和问题。它介于个人方法论、思想札记和创造者手册之间。

**状态**：v0.4，八篇核心文章 + 序言 + 结语 + 辅助资料

核心问题：
- 在身份之前，是什么先抓住了我？
- 一个问题如何从现象中慢慢长出来？
- 如何向智者学习，而不只是寻找答案？
- 作品和组织又如何反过来塑造我？
- 身份如何成为脚手架，而不是房子？
