# 参考文献

本小册子在写作过程中参考或借用了以下作品和思想资源。标注为"#参考"的表示正文中引用了具体概念或案例，标注为"#延伸"的表示与小册子主题相关的进一步阅读。

## 核心参考

戈夫曼，欧文.
- 《日常生活中的自我呈现》。1956。（#参考：拟剧论、前台/后台）
- 《精神病院》。1961。（#参考：全控机构）
- 《污名》。1963。（#参考：污名、身份管理）
- 《互动仪式》。1967。（#参考：面子工作、互动秩序）

布迪厄，皮埃尔.
- 《区隔》。1979。（#参考：品味、文化资本、惯习）
- 《实践感》。1980。（#参考：惯习、场域、实践理论）
- 《再生产》。1970，与让-克洛德·帕斯隆合著。（#参考：文化资本再生产）

米尔斯，C. 赖特.
- 《社会学的想象力》。1959。（#参考：个人困扰与公共议题）
- 《权力精英》。1956。（#参考：权力结构）

凯利，乔治.
- 《个人建构心理学》两卷本。1955。（#参考：个人建构、建构替代论）

西蒙，赫伯特.
- 《管理行为》。1947。（#参考：有限理性）
- 《组织》。1958，与詹姆斯·马奇合著。（#参考：组织行为）
