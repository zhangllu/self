# 学习智者的高阶认知：主动适应复杂社会

> 身份是脚手架，不是房子。

这是一本仍在生长中的小册子，讨论一个创造者、研究者或转型中的人，如何不急着给自己定位，而是先靠近那些反复抓住自己的现象、困惑和问题。

它介于个人方法论、思想札记和创造者手册之间。
