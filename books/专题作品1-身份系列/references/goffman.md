# Erving Goffman

## 为什么放进这本小册子

戈夫曼不是作为社会学知识点出现，而是作为"理论如何从长期凝视现象中长出来"的案例。

## 生平概览

| 年份 | 事件 |
|---|---|
| 1922 | 出生于加拿大阿尔伯塔，乌克兰犹太移民家庭 |
| 1939 | 进入曼尼托巴大学，最初学习自然科学 |
| 1943 | 在加拿大国家电影局工作 |

| 1949 | 进入芝加哥大学社会学系 |

| 1961 | 出版《精神病院》与《遭遇》 |

| 1967 | 出版《互动仪式》 |
| 1974 | 出版《框架分析》 |

## 核心概念

- **自我呈现** — 人在他人面前如何管理自己呈现出来的形象
- **拟剧论** — 用戏剧隐喻（前台、后台、观众、剧班）分析日常互动
- **印象管理** — 人如何控制别人对自己的印象
- **面子工作** — 维持社交场面中的体面和尊严
- **污名** — 社会标记如何影响一个人的身份和互动
- **全控机构** — 机构如何全面管理人的生活（如精神病院、监狱）
- **互动秩序** — 日常互动本身具有自己的结构，不能还原为宏观社会结构

## 思考风格

戈夫曼的思考方式有几个显著特征：

1. **现象优先**：他的理论不是从宏大问题出发，而是从细小、日常、容易被忽视的互动细节长出来的。
2. **概念发明**：他擅长为常见现象创造新概念，让读者"看见"原本看不见的东西。
3. **不追求体系**：他不像帕森斯或布迪厄那样试图建立统一理论体系，而是不断打开新现象域。
4. **写作风格**：他的文字敏锐、具体、甚至幽默，与社会学的学术正统保持距离。

## 与本小册子的关系

戈夫曼提醒我：

- 一个人不必先归入某个流派，而可以先忠于自己被抓住的现象。
- 概念不是空洞包装，而是从长期观察中长出来的。
- 日常互动不是小事——社会秩序就发生在一次寒暄、一场沉默、一次补救里。

## 与其他思想人物的关联

- **与布迪厄的对比**：布迪厄从阶层、品味和文化资本进入，更关注宏观不平等如何进入身体；戈夫曼从微观互动进入，更关注日常互动中的秩序和呈现。
- **与米尔斯的对比**：米尔斯从公共责任和权力结构进入；戈夫曼从个人互动和日常场景进入。
- **与凯利的对话**：凯利的个人建构理论强调人通过建构系统理解世界，戈夫曼则展示这些建构如何在互动中被呈现、维持和修补。
