# 专题作品3：给现代人的焦虑辨认地图

> 焦虑不只是要被消灭的症状。它也是一封混乱的来信。
> 只是我们需要学会分辨：这封信，是身体写来的，关系写来的，自由写来的，文化写来的，还是时代写来的。

讨论焦虑作为一种现代人的普遍经验——它不是需要被消灭的问题，而是一种可以被理解、被回应、被转化的信号。
