# George Kelly

## 为什么放进这本小册子

凯利的个人建构理论为"身份是脚手架，不是房子"提供了心理学层面的理论支撑。

## 生平概览

| 年份 | 事件 |
|---|---|
| 1905 | 出生于美国堪萨斯州的一个农民家庭 |
| 1926 | 获得帕克学院物理学和数学学士学位 |
| 1928 | 获得堪萨斯大学教育社会学硕士学位 |
| 1931 | 获得爱荷华大学心理学博士学位 |
| 1930s-1940s | 在堪萨斯州立学院任教，发展个人建构理论 |
| 1955 | 出版两卷本《个人建构心理学》，提出完整理论体系 |
| 1960s | 理论在全球范围内产生影响，应用于临床、教育、组织等领域 |
| 1967 | 去世 |

## 核心概念

- **个人建构** — 人通过"建构"（construct）来理解世界，建构是一组对立维度（如"安全-危险""开放-封闭"），人用它们来预测和解释经验。
- **建构替代论** — 一个人可以不断修正、替换、重组自己的建构系统。没有建构是永恒真理，只有适用与不适用的区别。
- **人是科学家** — 凯利的核心隐喻：每个人都像科学家一样，通过提出假设、检验预测、修正理论来理解世界。
- **焦虑** — 当一个人的建构系统无法处理新经验时产生的体验。
- **角色建构** — 人如何理解他人的建构系统，并在互动中调整自己的行为。

## 思考风格

1. **认识论立场独特**：凯利不把心理学建立在行为主义或精神分析的基础上，而是提出"人是意义建构者"的模型。
2. **开放系统**：他的理论不是封闭体系，而是鼓励每个人发展自己的建构方式。
3. **实践导向**：个人建构理论被广泛应用于心理治疗、教育、组织发展等领域。
4. **乐观**：他认为人不是被过去决定的，而是有能力重新建构自己的经验世界。

## 与本小册子的关系

凯利提醒我：

- 身份不是对"真实自我"的最终发现，而是一种当前阶段的建构。
- 如果某个身份不再适用，说明它完成了阶段性功能，需要被替换或升级。
- 人不必被过去的建构限制，可以重新建构对自己和世界的理解。
- "建构替代论"和"身份是脚手架"有深刻的呼应：身份可以作为工作假设，帮助行动，但必须允许被修正。

## 关键作品

- 《个人建构心理学》两卷本（1955）

## 与其他思想人物的关联

- **与戈夫曼的关联**：戈夫曼展示人如何在互动中呈现和维持自我，凯利则在认知层面解释人如何建构对自我和世界的理解。两人都反对把"真实自我"看作固定本质。
- **与小册子主题的关联**：第七篇"身份是脚手架，不是房子"的核心隐喻，与凯利的建构替代论高度一致。
