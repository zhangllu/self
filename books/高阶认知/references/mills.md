# C. Wright Mills

## 为什么放进这本小册子

米尔斯用来说明：同样是社会学，也可以从公共议题、权力结构和知识分子责任进入。

## 生平概览

| 年份 | 事件 |
|---|---|
| 1916 | 出生于美国德克萨斯州，中产阶级家庭 |
| 1939 | 获得德克萨斯大学学士学位 |
| 1941 | 获得威斯康星大学博士学位 |
| 1945 | 进入哥伦比亚大学社会学系任教 |
| 1951 | 出版《白领：美国中产阶级》 |
| 1956 | 出版《权力精英》，分析美国商业、政治和军事精英的权力集中 |
| 1959 | 出版《社会学的想象力》，成为社会学领域被引用最多的著作之一 |
| 1962 | 因心脏病去世，年仅46岁 |

## 核心概念

- **社会学的想象力** — 将个人困扰与公共议题连接起来的能力，是社会学思维的核心品质
- **个人困扰与公共议题** — 区分"个人层面的问题"和"社会结构层面的问题"的方法论工具
- **权力精英** — 商业、政治和军事领域的少数人掌握了不成比例的巨大权力
- **白领** — 新中产阶级在组织社会中的位置和困境
- **知识分子责任** — 知识分子有责任揭示权力结构，而非成为权力体系的附庸

## 思考风格

1. **公共导向**：他不认为社会学只是学术圈内的事，坚持面向公共读者写作。
2. **批判性**：他始终关注权力、不平等和社会结构，对美国社会的权力集中持尖锐批判立场。
3. **反对宏大理论**：在《社会学的想象力》中，他批评帕森斯的宏大理论和抽象经验主义，主张问题导向的研究。
4. **道德紧迫感**：他的写作有强烈的道德紧迫感和公共关怀。

## 与本小册子的关系

米尔斯提醒我：

- 个人的困惑和困扰背后往往有公共议题和社会结构的原因。
- 知识分子（包括创造者、写作者）有责任让更多人看见权力结构如何运作。
- 一个人的问题意识可以从公共责任和时代结构中生长出来。

## 关键作品

- 《白领：美国中产阶级》（1951）
- 《权力精英》（1956）
- 《社会学的想象力》（1959）

## 与其他思想人物的关联

- **与戈夫曼的对比**：戈夫曼关注微观互动，米尔斯关注宏观权力结构。但两人都反对学术体制的僵化，都坚持自己的写作风格。
- **与布迪厄的对比**：两人都关注权力和不平等，但米尔斯更直接批判权力精英，布迪厄更关注文化资本和象征权力在日常中的运作机制。
