# Herbert Simon

## 为什么放进这本小册子

西蒙用来说明：一个有创造力的思想家不只需要狭窄的专业化，也需要在不同的领域中持续转换视角。

## 生平概览

| 年份 | 事件 |
|---|---|
| 1916 | 出生于美国威斯康星州密尔沃基 |
| 1936 | 芝加哥大学政治学学士学位 |
| 1943 | 芝加哥大学政治学博士学位 |
| 1949 | 加入卡内基理工学院（后来的卡内基梅隆大学），参与创办工业管理研究生院 |
| 1958 | 与马奇合著《组织》，开创组织行为研究的认知传统 |
| 1969 | 出版《人工智能科学》 |
| 1975 | 获图灵奖（与纽厄尔共同获奖），因人工智能和认知科学的基础贡献 |
| 1978 | 获诺贝尔经济学奖，因经济组织的决策过程研究 |
| 1990s | 继续在认知科学、问题解决、科学发现等领域发表作品 |
| 2001 | 去世 |

## 核心概念

- **有限理性** — 人的决策不是完全理性的，而是受到信息、时间和认知能力的限制。人寻找"满意解"而非"最优解"。
- **问题解决** — 与纽厄尔共同提出"通用问题解决者"模型，认为问题解决是搜索问题空间的过程。
- **认知科学** — 西蒙是认知科学的核心创始人之一，主张可以用信息加工模型解释人类思维。
- **组织行为** — 组织如何影响个体的决策和行动，组织中的信息流和注意力分配。
- **科学发现** — 西蒙后期研究科学发现的认知机制，认为科学发现可以用计算模型模拟。

## 思考风格

1. **跨学科**：他是罕有的横跨政治学、经济学、心理学、计算机科学和管理学并做出重要贡献的学者。
2. **计算主义**：他坚信人的思维可以用信息加工模型来理解，人的认知和计算机的信息处理有相同的底层逻辑。
3. **问题驱动**：他不受学科边界限制，始终以问题（决策、解决问题、组织行为）为中心选择工具和语言。
4. **体系化**：他试图用统一的认知框架解释从个人决策到组织行为到科学发现的广泛现象。

## 与本小册子的关系

西蒙提醒我：

- 一个有创造力的思想者不必固守一个学科。问题本身就是最好的组织原则。
- 有限理性这个洞察也适用于身份：一个人不可能完全理性地规划自己的身份和路径，"满意解"可能比"最优解"更现实。
- 组织的核心功能之一就是管理注意力——这和组织如何塑造人的可能性直接相关。

## 关键作品

- 《管理行为》（1947）
- 《组织》（1958，与马奇合著）
- 《人工智能科学》（1969）
- 《人类问题解决》（1972，与纽厄尔合著）
- 《有限理性模型》（1982）

## 与其他思想人物的关联

- **与戈夫曼的关联**：两人都关注日常互动中的秩序，但西蒙关注组织结构中的决策和信息流，戈夫曼关注互动秩序中的自我呈现。
- **与凯利的关联**：两人都关心人如何处理信息、如何建构对世界的理解，但凯利来自临床心理学传统，西蒙来自认知科学和管理科学传统。
- **与跨学科实践的关联**：西蒙本人的学术生涯就是"不被学科限制"的典范，这与小册子第四篇"学科是入口，不是归宿"高度呼应。
