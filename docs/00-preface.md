# 序言：一本还在生长中的小册子

## 它从哪里来

这本小册子起源于一个简单的困惑：为什么我常常在刚被某个现象抓住时，就急着给自己一个身份？

不是"我没有方向"。恰恰相反，我被很多东西抓住。教育、社群、作品、组织、AI、人的可能性、系统的塑造力。但在这些关注背后，有一种反复出现的冲动：一旦我发现自己对什么有感觉，就会很快问自己——这代表我是谁？我是不是要成为某种人？我是不是应该有一个清晰的身份和产品？

这个困惑本身不是新的。很多人都在经历类似的拉扯：想专注，又怕太早定型；想开放，又怕太散；想定位，又怕定位太早把可能性封死了。

但我发现，我真正的困扰不在于"不知道方向"，而在于——我常常太早把方向压缩成身份。现象刚刚出现，我就想变成标签；困惑刚刚成形，我就想交出答案；关注刚刚聚集，我就想宣布事业。

后来，在和 AI 的持续对话中，我开始把这种拉扯写下来。一开始只是一些碎片式的观察和反思，慢慢地，它们聚成了七篇文章。这七篇文章不是在表达一个已经完成的答案，而是在记录一次尚未完成的思考过程。

这本小册子，就是这份记录。

## 它是什么

它介于个人方法论、思想札记和创造者手册之间。

说"个人方法论"，是因为它确实起源于我自己的困惑——我如何处理注意力、身份和作品的关系。

说"思想札记"，是因为它在不同地方借用了戈夫曼、布迪厄、米尔斯、凯利、西蒙等人的视角，但并不是为了做学术综述，而是为了让自己看见：我的困惑早已有人以不同方式回答过。

说"创造者手册"，是因为它最终想回应的不只是一个私人问题，而是很多正在做作品、正在找方向、正在犹豫要不要给自己一个身份的人，可能都会遇到的张力。

## 它不是什么

它不是职业定位指南。它不会教你如何找到"真正的热情"或"人生使命"。

它不是完整的学术综述。它对戈夫曼、布迪厄的使用是选择性的，服务于小册子的问题线索，而不是服务于学科完整性。

它不是思想人物传记。它提到戈夫曼的生命经历，不是为了写传记，而是为了说明"理论可以从长期凝视现象中长出来"。

它也不是一个已经完成的理论体系。它只是一个阶段性的写作——记录了我目前理解到什么程度，还有哪些东西没有完全想清楚。

## 适合谁读

如果你正在经历以下任何一种状态，这本小册子可能和你有关系：

- 你关注很多领域，却因此感到分散和焦虑。
- 你被一些人说是"跨领域"或"不聚焦"，但你知道自己底层有关联性。
- 你正在做一个需要长时间积累的作品，而社会希望你快点给出定位。
- 你感觉"个人品牌"在帮助你，也在绑架你。
- 你对身份这个词既需要又不信任。
- 你正在转型——从一种身份过渡到另一种，或者从单一身份进入更开放的自我理解。

你不必是写作者、研究者或创造者。这本小册子里的"作品"可以是你做的任何东西：一个项目、一个社群、一个产品、一个家庭、一种生活方式，甚至一种你自己还没有命名的实践。

## 怎么读

七篇文章有内部顺序——从"太早定位"开始，到"身份是脚手架"结束。如果你从头到尾读，会看到一条从现象到问题到作品到身份的生成线索。

但你也可以按需阅读：

- **如果你正在经历身份焦虑**：读 01、02、07
- **如果你正在做作品**：读 03、04、06
- **如果你关心思想人物的生成方式**：读 05，配合参考页
- **如果你想看到整个框架**：从头读到尾

每篇文章末尾都有 Changelog，记录它从初稿到当前版本的变化。你可以看见这些文章本身也在生长——这也是小册子主题的自我实践：作品不是一次定稿的，它在被写的过程中也在塑造写作者。

## 当前状态

这本小册子目前是 v0.2，处于"持续生长中"的状态。

- 七篇核心文章已经完成初稿。
- 现象清单、材料库、问题清单正在生长。
- 参考文献正在积累。
- 后续可能增加新的章节，也可能对已有章节做大幅修订。

如果你读到这里，欢迎通过 GitHub Issue 提交你的现象记录、理论补充或阅读感受。这个仓库本身也在实践它所讨论的事情：作品和组织也在反过来塑造创作者。
