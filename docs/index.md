# 从现象到身份：七篇文章的结构

这本小册子的七篇文章，内部有一条从"内"到"外"再到"合"的线索。

## 第一篇到第三篇：向内看

前三篇处理的是个人层面的问题。

- **第一篇** 诊断核心问题：太早定位。
- **第二篇** 往前追问：抓住我们的起点不是问题，而是现象。
- **第三篇** 处理一个派生问题：关注很多怎么办？提出了关注分层。

核心主张：先看见自己如何被现象抓住，不要急着命名。

## 第四篇到第六篇：向外看

中间三篇转向外部世界。

- **第四篇** 寻找已有回答：学科、组织和作品已经在回应我们关心的问题。不要急着原创，先画出已有回答地图。
- **第五篇** 用思想人物验证：戈夫曼、布迪厄、米尔斯的生成路径说明，理论是从长期凝视中长出来的，而不是从身份野心开始的。
- **第六篇** 回应一个补充：进入外部世界不只是单向学习——作品和组织也在反过来塑造我们。

核心主张：带着现象进入世界，但也会被世界塑造。

## 第七篇到结语：收束

- **第七篇** 提出核心隐喻：身份是脚手架，不是房子。
- **结语** 梳理线索，承认未回答的张力，指向下一步。

核心主张：身份是沉淀，不是起点。

## 关系图

```
现象（抓住我）→ 困惑（反复出现）→ 问题（成形）→ 进入（学科/组织/作品）→ 被塑造 → 身份沉淀
    ↑                                                                              |
    └────────────────────── 循环：新现象继续出现 ──────────────────────────────────┘
```
