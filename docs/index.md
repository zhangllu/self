# 从现象到身份：八篇文章的结构

这本小册子的八篇文章（外加序言与结语），内部有一条从"内"到"外"再到"合"的线索。

## 第一篇到第三篇：向内看

前三篇处理的是个人层面的问题。

- **第一篇** 诊断核心问题：太早定位。不是没有方向，而是太早把方向压缩成身份。
- **第二篇** 往前追问：抓住我们的起点不是问题，而是现象。先保护现象，再提出问题。
- **第三篇** 处理派生问题：关注很多怎么办？提出六层关注分层，不是所有东西都有同等分量。

核心主张：先看见自己如何被现象抓住，不要急着命名。

## 第四篇到第六篇：向外看

中间三篇转向外部世界，引入研究方法。

- **第四篇** 引入核心方法：向智者学习——不是把他们当答案库，而是学习他们的认知方式。
- **第五篇** 拓展研究地图：除了智者，学科、组织、作品、案例、社群和行动实验也在以不同方式回答同一个现象。先做已有回答地图。
- **第六篇** 回应一个补充：进入外部世界不只是单向学习——作品、组织、社群这些形式也在反过来训练、限制、校准和塑造我们。

核心主张：带着现象进入世界，但也会被世界塑造。

## 第七篇到结语：回到身份

最后两篇收束前面的线索。

- **第七篇** 提出核心隐喻：身份是脚手架，不是房子。它不是起点，而是工作假设和阶段性结构。
- **结语** 梳理完整线索，总结身份如何从现象、问题、分量、智者、已有回答、作品和组织的持续互动中慢慢沉淀。

核心主张：身份是沉淀，不是起点。

## 关系图

```
现象 → 困惑 → 问题 → 向智者学习 → 已有回答地图（学科/组织/作品/实验） → 进入形式（被塑造） → 身份沉淀
  ↑                                                                                                   │
  └───────────────────────────── 循环：新现象继续出现 ───────────────────────────────────────────────┘
```
