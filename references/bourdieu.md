# Pierre Bourdieu

## 为什么放进这本小册子

布迪厄用来说明：同样进入社会学，不同人的问题敏感性和生命经历会生成不同的进入口。

## 生平概览

| 年份 | 事件 |
|---|---|
| 1930 | 出生于法国南部比利牛斯地区的工人阶级家庭 |
| 1950s | 进入巴黎高等师范学院，接受精英教育 |
| 1958 | 出版《阿尔及利亚的社会学》，基于在阿尔及利亚战争期间的田野研究 |
| 1964 | 出版《继承人》，讨论教育系统中的不平等再生产 |
| 1970 | 出版《再生产》，系统阐述文化资本和惯习理论 |
| 1979 | 出版《区隔》，基于大规模调查分析品味如何作为阶层标志 |
| 1980 | 出版《实践感》，系统阐述惯习、场域、实践理论 |
| 1990s | 积极参与公共事务，成为法国最重要的公共知识分子之一 |
| 2002 | 去世 |

## 核心概念

- **惯习** — 内化于身体中的感知、思考和行动倾向系统，既被社会结构塑造，又反过来生成实践
- **场域** — 具有特定规则和权力结构的社会空间，行动者在其中争夺位置和资源
- **文化资本** — 知识、品味、教育资历等非经济资本，与阶层再生产密切关联
- **区隔** — 社会群体通过品味和文化实践区分自己，这种区分本身就在维持阶层结构
- **象征权力** — 通过语言、分类系统和符号来维护或挑战社会秩序的权力
- **社会再生产** — 社会不平等如何通过教育、家庭和文化机制代际传递

## 思考风格

1. **体系化**：布迪厄追求建立统一的理论语言（惯习、场域、资本），试图超越主观/客观、个人/社会的二元对立。
2. **经验与理论并重**：他的理论建立在大量经验研究基础上（阿尔及利亚田野、法国教育系统、博物馆观众调查）。
3. **批判视角**：他始终关注权力、不平等和社会再生产，把学术工作本身也纳入批判。
4. **概念密度高**：他的写作风格以概念密度高、句法复杂著称，阅读门槛较高。

## 与本小册子的关系

布迪厄提醒我：

- 限制、阶层和教育经历不只是障碍，也可能成为一个人看见世界的独特位置。
- 社会不平等不只是外部条件，它会进入身体、品味和日常判断。
- 任何组织或场域都不是中性空间，权力在其中运作。

## 关键作品

- 《继承人》（1964）
- 《再生产》（1970）
- 《区隔》（1979）
- 《实践感》（1980）
- 《国家精英》（1989）

## 与其他思想人物的关联

- **与戈夫曼的对比**：两人都关注社会秩序如何维持，但戈夫曼更关注微观互动中的秩序，布迪厄更关注宏观不平等如何通过教育和文化再生产。
- **与米尔斯的对比**：两人都关注权力，但米尔斯更关注权力精英和公共责任，布迪厄更关注日常实践中的象征权力和文化再生产。
